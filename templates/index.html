Goal of the Prediction

The goal of this prediction tool is to estimate the type of Olympic medal (Bronze, Silver, or Gold) a team is most likely to win based on the discipline and specific event they participate in.

How It Works

User Input: Users provide the Discipline and Event IDs in the form fields.

Prediction: The FastAPI backend uses a trained Random Forest model to predict the type of medal (Bronze, Silver, or Gold) the team might win based on the provided input.

Output: The predicted medal type is displayed on the same page below the form.

Sample Inputs:

Example 1: Discipline: 1, Event: 100 (Athletics, 100m Sprint)

Example 2: Discipline: 2, Event: 200 (Swimming, 200m Freestyle)

Example 3: Discipline: 3, Event: 300 (Gymnastics, Floor Exercise)
